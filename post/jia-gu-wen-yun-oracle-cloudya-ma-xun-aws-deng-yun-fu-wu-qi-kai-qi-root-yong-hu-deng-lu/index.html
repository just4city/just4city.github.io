甲骨文云Oracle Cloud/亚马逊AWS等云服务器开启root用户登录

2019-11-14 10:50

一些云服务商默认禁止了root用户登录, 现在分享一下开启root用户登录的方法.
以oracle cloud的centos系统为例:

ssh opc@你的ip地址

登录服务器

sudo -i

切换root用户

#把ssh-rsa之前的都删除
vi .ssh/authorized_keys

编辑ssh配置文件, 找到PermitRootLogin, 把前面的#去掉

vi /etc/ssh/sshd_config
PermitRootLogin yes

重启ssh服务

service sshd restart

更改密码

echo root:your_password |sudo chpasswd root
